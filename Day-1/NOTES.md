# 📝 Python Day 1 Notes

## What is Python?

Python is a high-level, interpreted, general-purpose programming language. It is easy to learn and widely used in modern software development.

---

## Key Features

- Easy to Read
- Easy to Learn
- Open Source
- Cross Platform
- Interpreted Language
- Large Standard Library

---

## Applications of Python

- Artificial Intelligence
- Machine Learning
- Data Science
- Web Development
- Automation
- Cybersecurity

---

## print() Function

Displays output on the screen.

### Syntax

```python
print("Hello, World!")
```

---

## input() Function

Takes input from the user.

### Syntax

```python
name = input("Enter Name: ")
```

---

## Comments

Comments improve code readability and are ignored during program execution.

### Syntax

```python
# This is a comment
```

---

## Key Points

- Python is case-sensitive.
- Indentation is important.
- Variables can store user input.
- `input()` always returns a string.
- Use `int()` to convert numeric input into an integer.

---

## Common Mistakes

- Forgetting quotation marks.
- Incorrect indentation.
- Misspelling `print`.
- Forgetting `int()` while taking numeric input.

---

## Quick Revision

- High-level language
- Interpreted language
- `print()` → Output
- `input()` → Input
- `#` → Comment
- `int()` → Integer Conversion
