# 📝 Python Day 2 Notes

## Variables

A variable is a named container used to store data in memory.

### Example

```python
name = "Aditya"
age = 19
```

---

## Variable Naming Rules

- Use meaningful names.
- Variable names are case-sensitive.
- Must begin with a letter or underscore (`_`).
- Cannot start with a number.
- Do not use Python keywords as variable names.

---

## Python Data Types

| Data Type | Example |
|-----------|---------|
| `str` | `"Aditya"` |
| `int` | `19` |
| `float` | `9.2` |
| `bool` | `True` |

---

## type() Function

Used to check the data type of a variable.

### Example

```python
print(type(age))
```

---

## Type Conversion

Convert one data type into another.

### String → Integer

```python
number = int("25")
```

### Integer → Float

```python
marks = float(95)
```

### Float → Integer

```python
height = int(187.5)
```

---

## Key Points

- Variables store values.
- Every value has a data type.
- `type()` identifies the data type.
- Use type conversion when required.
- Choose meaningful variable names.

---

## Common Mistakes

- Using Python keywords as variable names.
- Starting variable names with numbers.
- Forgetting type conversion.
- Using unclear variable names.

---

## Quick Revision

- Variable → Stores data
- `str` → Text
- `int` → Integer
- `float` → Decimal number
- `bool` → True/False
- `type()` → Checks data type
